# Pickton Storage — Static Website

This is a simple static website for **Pickton Storage** (Pickton, TX).

## Pages
- `index.html` — Home
- `units.html` — Unit details + pricing
- `signup.html` — Signup request form (static + Netlify-compatible)
- `porch-sittin.html` — “Porch Sittin” daily story page
- `shop.html` — Merch (print-on-demand) placeholder links
- `contact.html` — Contact + Facebook link

## Quick deploy (recommended: Netlify)
1. Create a new site on Netlify.
2. Drag-and-drop the folder `pickton-storage-site/` into Netlify, or connect a git repo.
3. (Optional) Enable **Forms** in Netlify to receive signup submissions.
   - The form is already marked up for Netlify.
   - Add notification email: **1mscottexas@gmail.com**.

## Daily story updates (Porch Sittin)
Edit:
- `content/porch-sittin.json`

Update the `date`, `title`, and `body` fields. The page auto-renders it.
